Summary: Global Weather Plots



Visualizations



The purpose of this project:
1) Analyze weather patterns as we get closer to the equator.
2) Utilize the OpenWeatherMap API and retrieve data to analyze a dataset over 500 cities.
3) Analyze established dataset and utilize Matplotlib to plot various weather patterns such as temperature, cloudiness, wind-speed and humidity based on the latitude .
4) Analyze established dataset and utilize Matplotlib to plot various weather patterns such as temperature, cloudiness, wind-speed, and humidity in terms of longitude vs latitude, globally.
5)This project provides visualization of the global weather data analysis.
